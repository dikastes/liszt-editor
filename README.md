# liszt_editor
This is a prototype of the data input mask for the Liszt Catalog Raisonn√©

## How to install

Make shure python or python3 is installed. Run the install script: 

```{bash}
# ./install.bash
```

## Run the server

Run the server from the root directory of liszt-editor:

```{bash}
# bin/python3 manage.py runserver
```

Alternatively run: 
```{bash}
# source bin/activate
# python3 manage.py runserver
```
